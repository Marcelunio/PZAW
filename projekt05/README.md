# Obiektywnie

Forum internetowe o Aparatach i urządzeniach  

## Instalacja  

Po zpullowaniu kodu źródłowego użyc następujących komend:  

```bash
npm install  
node --run initialize  
```

## Uruchamianie servera  
Aby uruchomic server wystarczy użyc komendy  

```bash
node --run start  
```

### Stos technologii:  
    1.Javascript  
    2.Node.js  
        * Express  
        * Ejs 
        * Argon2  
        * cookie-parser
